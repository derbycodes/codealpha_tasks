CodeAlpha Tasks

A collection of small Python projects completed as part of the CodeAlpha internship program. Each task focuses on a different set of core programming concepts from control flow and functions to file handling and automation.

📁 Projects
🎮 Hangman Game (hangman.py)

A console-based word-guessing game. The player guesses letters one at a time, with a limited number of incorrect guesses allowed before the game ends. Concepts: random, loops, if-else, strings, lists

💰 Stock Tracker (stock_tracker.py)

Calculates the total value of a simple stock portfolio based on user input and a hardcoded price list, with the option to save the summary as a .txt or .csv file. Concepts: dictionaries, input/output, arithmetic, file handling

💬 Rule-Based Chatbot (chatbot.py)

A simple chatbot that replies to a fixed set of greetings and phrases using pattern matching. Concepts: if-elif, functions, loops, input/output

🗂️ JPG File Organizer (jpg_organizer.py)

Automates a small real-life task — scanning a folder and moving all .jpg/.jpeg files into a separate destination folder, safely avoiding overwriting files with duplicate names. Concepts: os, shutil, file handling
